Impact Analysis:

    🔑 MINT New Capability

    MINT creates a new Golden Token in the next free Namespace slot and returns it in CR0. The GT remains accessible until RETURN releases the register.

    MINT will:
    • Allocate next free Namespace offset
    • Create 3-word Namespace entry (W1, W2, W3)
    • Generate GT with source permissions
    • Load new GT into CR0

    Note: To persist the GT beyond RETURN, use SAVE to add it to a C-List programmatically.
