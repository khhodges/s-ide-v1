In 1843, Ada Lovelace wrote the first computer program — an algorithm for the Analytical Engine using symbolic notation, one operation per line. Her Note G computed the Bernoulli numbers, and it was the first first-class program: it passed functions as arguments, not just values. That is what lambda calculus does — and what this prompt does. Type let x = 2 + 3 and you are writing what Ada wrote. To see her original algorithm as Church Machine code, click Code and select the Bernoulli example.
